# crz-imafooler
This is an All in One battery controller hacking tool for a Honda CR-Z Battery Pack.  
Features will include: 

- [] IMA MITM
- [] TPS Based Voltage manipulation
- [] TPS Based Current manipulation
- [] Setting SoC based on Voltage

Currently WIP :D
